# Knobase App Landing

Landing page for Knobase (Human-AI Collaboration Workspace)

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript
- Framer Motion (animations)
- Lucide React (icons)

## Design System

Minimal, clean aesthetic inspired by Notion/Linear:
- Pure white backgrounds
- Subtle 1px borders
- Generous whitespace
- Typography-focused
- Purple accent (#650BD8)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Connected to Vercel. Push to main to deploy.

```bash
vercel --prod
```
